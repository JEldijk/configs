# configs

## get configs

1. clone this repo
2. create links for all files:
   1. `ln -s ~/git/private/configs/.p10k.zsh ~/.p10k.zsh-backup`
   2. `ln -s ~/git/private/configs/.gitconfig.private ~/git/private/.gitconfig.private`
